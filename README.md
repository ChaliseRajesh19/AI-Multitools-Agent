# AI Tool Calling Assistant

## Features
- Multi-tool calling (calculator, weather, currency)
- Automatic tool selection by LLM
- Tool execution loop
- Structured JSON outputs

## Tech Stack
- Python
- LLM API (Groq/OpenAI)
- Tool Calling Architecture

## How it works
User → LLM → Tool selection → Execution → Response

## Run project
pip install -r requirements.txt
python app.py
